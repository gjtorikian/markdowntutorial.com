Links

Wir werden nun lernen, wie man Links zu anderen Webseiten im World Wide Web erstellt.

Es gibt zwei verschiedene Link-Stile in Markdown, die jedoch beide auf die gleiche Weise dargestellt werden. Der erste Link-Stil wird Inline-Link genannt. Um einen Inline-Link zu erstellen, wird der Linktext in eckige Klammern ( [ ] ) und dann der Link in Klammern ( ( ) ) eingeschlossen. Um zum Beispiel einen Hyperlink zu www.github.com zu erstellen, mit einem Linktext, der lautet: „Besuchen Sie GitHub!“, würden Sie folgendes in Markdown schreiben: [Besuchen Sie GitHub!](www.github.com).

Erstellen Sie in dem Feld unten einen Link zu www.google.com, mit einem Linktext, der lautet "Suche danach."

Gute Arbeit!

Sie können Linktexte hervorheben, wenn Sie möchten. Im untenstehenden Kasten machen Sie den Satz "wirklich, wirklich" fett und verlinken den gesamten Satz mit www.dailykitten.com. Achten Sie darauf, dass die fettgedruckte Formulierung innerhalb der Klammern des Linktextes steht.

Fantastisch!

Der andere Link-Stil wird als Referenz-Link bezeichnet. Wie der Name schon sagt, ist der Link eigentlich ein Verweis auf eine andere Stelle im Dokument. Hier ist ein Beispiel dafür, was wir meinen:

     Hier ist [ein Link zu einer Webseite][eine-webseite].
     Hier ist [noch einen Link][ein-anderer-link]. 
     Und jetzt zurück zu [dem ersten Link][eine-webseite].

     [eine-webseite]: www.github.com
     [ein-anderer-link]: www.google.com

Die "Referenzen" oben sind das zweite Set von Klammern: [eine-webseite] und [ein-anderer-Link]. Am Ende eines Markdown-Dokuments werden diese Klammern als richtige Links zu externen Webseiten definiert. Ein Vorteil des Referenzlink-Stils ist, dass mehrere Links zum gleichen Ort nur einmal aktualisiert werden müssen. Zum Beispiel, wenn wir beschließen, dass alle [eine-webseite] Links woanders hinführen sollen, müssen wir nur den einzelnen Referenzlink ändern.

Referenzlinks erscheinen nicht im gerenderten Markdown. Sie definieren diese, indem Sie denselben Tag-Namen in Klammern setzen, gefolgt von einem Doppelpunkt und dem Link.

Im unteren Feld haben wir angefangen, einige Referenzlinks zu schreiben. Vervollständigen Sie diese! Nennen Sie den ersten Referenz-Tag "ein lustiger Ort" und verlinken Sie ihn mit www.zombo.com; machen Sie den zweiten Link zu www.stumbleupon.com.

Jetzt wissen Sie genau, wie man Links in Markdown erstellt!
