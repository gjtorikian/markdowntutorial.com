Links

Agora vamos aprender como fazer links para outros sites na World Wide Web.

Existem dois tipos diferentes de links no Markdown, mas ambos são renderizados da mesma maneira. O primeiro estilo de link é chamado de link inline. Para criar um link inline, você envolve o texto do link entre colchetes ( [ ] ), e depois envolve o link entre parênteses ( ( ) ). Por exemplo, para criar um hyperlink para www.github.com, com um texto de link que diz, Visite o GitHub!, você escreveria isso no Markdown: [Visite o GitHub!](www.github.com).

Na caixa abaixo, faça um link para www.google.com, com o texto do link que diz "Busque por isso."

Bom trabalho!

Você pode adicionar ênfase ao texto do link, se quiser. Na caixa abaixo, deixe a frase "muito, muito" negrito e faça com que toda a frase seja um link para www.dailykitten.com. Você vai querer garantir que a formatação em negrito ocorra dentro dos colchetes do texto do link.

Isso é tudo para escrever links inline.

O outro tipo de link é chamado de link de referência. Como o nome sugere, o link é na verdade uma referência a outro lugar no documento. Aqui está um exemplo do que queremos dizer:

     Aqui está [um link para outro lugar][outro lugar].
     Aqui está [mais um link][outro-link].
     E agora de volta ao [primeiro link][outro lugar].

     [outro lugar]: www.github.com
     [outro-link]: www.google.com

As "referências" acima são o segundo conjunto de colchetes: [outro lugar] e [outro-link]. Na parte inferior de um documento Markdown, esses colchetes são definidos como links adequados para sites externos. Uma vantagem do estilo de link de referência é que vários links para o mesmo lugar só precisam ser atualizados uma vez. Por exemplo, se decidirmos fazer com que todos os links [outro lugar] vão para outro lugar, só precisamos mudar o link de referência único.

Links de referência não aparecem no Markdown renderizado. Você os define fornecendo o mesmo nome de tag envolto em colchetes, seguido por dois pontos, seguido pelo link.

Na caixa abaixo, começamos a escrever alguns links de referência. Você precisará terminá-los! Chame a primeira tag de referência de "um lugar divertido" e faça um link para www.zombo.com; faça o segundo link para www.stumbleupon.com.

Agora você sabe como fazer links no Markdown!
