Markdown 是一種用來撰寫網頁內容的方式。它是以「純文字格式」來書寫，純文字常用於傳送訊息，由字母、常見的符號如星號 ( * 或反引號( ` )……等字元組成。

不像傳統繁雜的文字處理軟體，用 Markdown 撰寫的內容可以輕易地在電腦、手機、人們之間分享。 Markdown 因此很快地成為學者、科學家、作家……等的寫作規範。 而一些網站如 GitHub、 reddit 也已經讓使用者用 Markdown 留言。

以 Markdown 來格式化文本是很簡單的，因為它省略了花俏的設定如：字型大小、顏色。你只需掌控文本本身的顯示類型，如：加粗、插入標題、組織清單。

只要十分鐘，你就能學會 Markdown！
